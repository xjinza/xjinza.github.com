Commonly used regular expression

Regular expressions are powerful tools for pattern matching on strings. Commonly used for form validation, filtering, and replacement of special characters.

First, regular expression rules

  • Metacharacters
    .: Find a single character, except newline or line terminator.
    \w: Find a word character.
    \W: Find a non-word character.
    \d: Find a digit.
    \D: Find a non-digit character.
    \s: Find a whitespace character.
    \S: Find a non-whitespace character.

  • Brackets
    [abc]: Find any character between the brackets
    [^abc]: Find any character NOT between the brackets
    [0-9]: Find any character between the brackets (any digit).
    [a-z]: Find any character from lowercase a to lowercase z.
    [A-Z]: Find any character from uppercase A to uppercase z.
    (red | blue | green): Find any specified options.

  • Quantifiers
    n+: Matches any string that contains at least one n.
    n*: Matches any string that contains zero or more occurrences of n.
    n?: Matches any string that contains zero or one occurrences of n.
    n{x}: Matches any string that contains a sequence of X n’s.
    n{x, y}: Matches any string that contains a sequence of X to Y n’s.
    n$: Matches any string that ends with n.
    ^n: Matches any string that beginns with n.

Second, commonly used regular expression rules

  • Check the expression of the number
    1.Non-negative integer: /^\d+$/
    2.positive integer: /^[1-9]\d*$/
    3.Up to two decimal digits: /^\d+(.\d{1,2})?$/

  • Check the character expression
    1.26 letters consisting of letters: /^[A-z]+$/
    2.Chinese characters: /^[\u4e00-\u9fa5]+$/
    3.At the beginning of the letter, the length between 6-18, can only contain characters, numbers and underscores: /^[A-z]\w{6,18}$/ // commonly used to verify the user name, password

  • Special requirements expression

  1. Email address:
    /^(([^<>()[]\.,;:\s@”]+(.[^<>()[]\.,;:\s@”]+)*)|(“.+”))@(([[0-9]{1,3}.[0-9]{1,3}.[0-9]{1,3}.[0-9]{1,3}])|(([a-zA-Z-0-9]+.)+[a-zA-Z]{2,}))$/

  2. Url address:
    /^(http|ftp|https):\/\/[\w-]+(.[\w-]+)+([\w.,@?^=%&:\/~+#-]*[\w@?^=%&\/~+#-])?$/

  3. Mobile phone number:
    /^1(3|4|5|7|8)\d{9}$/
  4. Foreign mobile phone number:
    /^(+\d{1,3}[- ]?)?\d{10}$/
  5. The first and last whitespace characters:
    /(^\s+)|(\s+$)/
  6. Filter all non-word and whitespace characters:
    mystring.replace(/[^\w\s]/gi,’’)
  7. Replace html special characters &> <”:
    mystring
    .replace(/&/g, “&”)
    .replace(/>/g, “>”)
    .replace(/</g, “<”)
    .replace(/“/g, “””);
  8. Replace the special characters in the json string:
    myJSONString.replace(/\n/g, “\n”)
    .replace(/\‘/g, “\‘“)
    .replace(/\“/g, ‘\“‘)
    .replace(/\&/g, “\&”)
    .replace(/\r/g, “\r”)
    .replace(/\t/g, “\t”)
    .replace(/\b/g, “\b”)
    .replace(/\f/g, “\f”);
